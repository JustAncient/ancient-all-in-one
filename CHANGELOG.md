# Ancient All-in-One Changelog

All notable changes to this project should be documented here.

## [0.1.0] - Initial release

- Created the initial desktop tracker framework.
- Added dynamic app naming and GitHub release update checks.
- Renamed the repo and internal package to Ancient All-in-One.
- Added logging, diagnostics, app-data storage, and corrupt-data recovery.
- Added safety foundations for backups, validation, migrations, settings, import/export, and module registry.
